# **App Name**: LoveSpark Calculator

## Core Features:

- Name Input: Allows users to input two names for compatibility calculation.
- Compatibility Calculation: Calculates a compatibility percentage based on the input names using a proprietary algorithm.
- Playful Message Generation: Generates a playful message alongside the compatibility percentage, offering fun insights into the relationship; the system acts as a tool and uses contextual data to incorporate user details.
- Result Saving: Saves the calculated compatibility percentage and playful message to Firebase Firestore along with input names.
- Calculation History: Allows users to view their past compatibility calculations stored in Firebase Firestore.

## Style Guidelines:

- Primary color: Vibrant pink (#F472B6) to evoke feelings of love and energy.
- Background color: Light pink (#FCE4EC), a desaturated version of the primary color, for a soft backdrop.
- Accent color: Purple (#A855BA), an analogous color to pink, adding contrast and visual interest.
- Font: 'Belleza' sans-serif for headlines and short chunks of text; 'Alegreya' serif for longer body text.
- Use heart and love-themed icons throughout the app.
- Include playful animations, such as sparkling effects, during the calculation and result display.
